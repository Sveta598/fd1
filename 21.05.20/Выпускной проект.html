Тема и содержание выпускного проекта

Сайт для выпускного проекта будет сделан на тему:

Спорт для детей

    Сайт будет включать в себя 4 типа страниц:
  1. Главная страница;
  2. Страницы с тематическими статьями про спорт для детей (8 страниц). Страницы будут сделаны по одному макету, различаться будут только названия статей, изображения и текст статей;
  3. Страницы с перечислением спортивных секций для детей по разным видам спорта (5 страниц). Страницы будут сделаны по одному макету, различаться будут только названия секций, изображения и текст;
  4. Страница со слайдером фотографий детей, занимающихся спортом.

Описание главной страницы

    Главная страница будет включать в себя:
  • Header, который будет содержать название сайта с логотипом, а также слайдер фотографий на заднем фоне;
  • Навигацию, которая будет вести на страницы с перечислением спортивных секций для детей по пяти видам спорта: плавание, баскетбол, гимнастика, теннис, шахматы;
  • Основную часть, которая будет состоять из трех колонок;
  • Footer, который будет содержать ссылку на авторское право и кнопки социальных сетей.
    Основная часть будет состоять из следующих колонок:
  • Левая боковая панель, содержащая перечень 8 тем, по которым будут написаны тематические статьи, анонсированные в основной части сайта: спорт для самых маленьких, баскетбол, бокс, плавание, велоспорт, шахматы, питание, спортивная экипировка. Эти темы будут ссылками, ведущими на соответствующие страницы с тематическими статьями. Под перечнем тем будет находиться кнопка-ссылка на предыдущие темы, так как предполагается возможность размещения на сайте статей по новым темам с перенесением предыдущих тем на другую страницу;
  • Основная часть, содержащая анонс статей (8 статей) по вышеприведенным темам в формате фотография + заголовок статьи и в форме ссылок, которые будут вести на соответствующие страницы с тематическими статьями. Что касается заголовков, то это будут не названия тем, а уже названия конкретных статей. Внизу под анонсом статей будет содержаться кнопка-ссылка на предыдущие статьи, так как предполагается возможность размещения на сайте новых статей с перенесением ранее написанных статей на другую страницу.
  • Правая боковая панель, содержащая ссылку на страницу со слайдером фотографий детей, занимающихся спортом;

На всех типах страниц сайта (4 типа) будет содержаться навигация по сайту, т.е. ссылки на страницы с перечислением детских спортивных секций, ссылки на страницы с тематическими статьями по темам и по названиям статей, ссылка на страницу со слайдером фотографий, а также кнопки социальных сетей. На разных типах страниц навигация и кнопки социальных сетей будут стилизовваны по-разному. Кнопки социальных сетей, логотип, а также другие иконки будут сделаны с помощью иконочных шрифтов или спрайтов. Возможно, некоторые изображения на страницах с перечислением спортивных секций будут сделаны с помощью спрайтов или средств CSS (градиентов, SVG и т.д.). Страницы сайта будут сверстаны с помощью Flex-технологии или Grid-технологии. Страницы сайта будут иметь резиновый дизайн и будут адаптированы под мобильные устройства. В частности, для мобильных версий будет изменен порядок расположения блоков на страницах.
